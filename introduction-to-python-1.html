Lecture: Working with Python Variables

In Python, variables are essential for storing and working with data. Let's explore some common variable operations and data types:

Variable Assignment

You can assign values to variables using the = operator:


                message = "Hello, Python!"
                number = 42
                pi = 3.14159

Variable Types

Python supports various data types:

  • Strings (str): Used for text, enclosed in single or double quotes.
  • Integers (int): Used for whole numbers.
  • Floats (float): Used for decimal numbers.
  • Booleans (bool): Used for true or false values.
  • Lists (list): Used to store collections of items.
  • Dictionaries (dict): Used to store key-value pairs.

Operations on Variables

You can perform various operations on variables, such as arithmetic calculations, string concatenation, and more:


                # Arithmetic operations
                a = 5
                b = 3
                sum_result = a + b
                difference = a - b
                product = a * b
                quotient = a / b

                # String concatenation
                first_name = "John"
                last_name = "Doe"
                full_name = first_name + " " + last_name

                # List operations
                fruits = ["apple", "banana", "cherry"]
                fruits.append("date")
                fruits.remove("banana")

                # Dictionary operations
                student = {"name": "Alice", "age": 25}
                student["grade"] = "A"
