Python Assignment 3

Assignment

Write a Python program that does the following:

  1. Creates a variable called score and assigns a numeric score to it (e.g., 85).
  2. Uses an if-elif-else statement to check the value of score and print a corresponding message:
    • If score is greater than or equal to 90, print "A: Excellent."
    • If score is between 80 and 89 (inclusive), print "B: Very Good."
    • If score is between 70 and 79 (inclusive), print "C: Good."
    • If score is between 60 and 69 (inclusive), print "D: Needs Improvement."
    • If score is less than 60, print "F: Fail."
  3. Modify the value of score to test different cases.
