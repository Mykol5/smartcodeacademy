Theoretical Description

HTML (Hypertext Markup Language) is the standard markup language used to create web pages. It provides a structured way to organize content on the web. HTML uses a system of tags to format text, images, links, and other media. Let's explore some essential HTML tags and concepts:

HTML Document Structure

An HTML document consists of the following components:

  • HTML Tag: Defines the document type and serves as the root element.
  • Head Section: Contains meta-information about the document, such as the title, character set, and linked stylesheets.
  • Body Section: Contains the visible content of the web page.

Common HTML Tags

HTML tags are used to structure and format content. Here are some common tags:

  • <h1> to <h6>: Define headings of varying levels.
  • <p>: Create paragraphs of text.
  • <a>: Create hyperlinks to other web pages.
  • <img>: Embed images into the page.
  • <ul> and <ol>: Define unordered and ordered lists.
  • <li>: Define list items.

Example

Here's an example of a simple HTML document:

                    
            <!DOCTYPE html>
            <html>
            <head>
                <title>My Web Page</title>
            </head>
            <body>
                <h1>Welcome to My Web Page</h1>
                <p>This is a sample web page created with HTML.</p>
                <a href="https://www.example.com">Visit Example.com</a>
                <img src="image.jpg" alt="An image">
            </body>
            </html>
