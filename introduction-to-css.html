Introduction to CSS

Cascading Style Sheets (CSS) are a fundamental part of web development. They allow us to control the presentation and layout of our web pages, making them visually appealing and user-friendly. There are several types of CSS, and they can be applied to an HTML document in various ways:

Types of CSS

  • Inline CSS: You can apply CSS styles directly to HTML elements using the "style" attribute. For example:
    •                                 
                <p style="color: blue; font-size: 16px;">This is a styled paragraph.</p>
  • Internal CSS: You can define CSS rules within the <style> element in the HTML document's <head> section. For example:
    •                                 
                <head>
                    <style>
                        h1 {
                            color: blue;
                        }
                    </style>
                </head>
  • External CSS: The most common approach is to create a separate CSS file (e.g., styles.css) and link it to the HTML document. For example:
    •                                 
                <head>
                    <link rel="stylesheet" href="styles.css">
                </head>

Each type of CSS has its advantages and use cases. Inline CSS is useful for quick styling of individual elements, internal CSS allows for document-specific styles, and external CSS promotes maintainability by separating content from presentation.

Class Example

Here's an example of how to create a basic CSS rule:

                    
/* Select all headings and change their text color to blue */
h1, h2, h3 {
    color: blue;
}

/* Set the background color of the page to light gray */
body {
    background-color: #f2f2f2;
}

/* Increase the font size of paragraphs to 16px */
p {
    font-size: 16px;
}
