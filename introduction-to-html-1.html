Forms and Input Elements

HTML allows you to create interactive forms that users can fill out. Forms are crucial for various online applications, from simple contact forms to complex user registrations. Here are some key form-related elements:

<form> Element

The <form> element is used to create an HTML form. It defines where user input will be collected and sent to a server for processing. It can include various input elements such as text fields, checkboxes, radio buttons, and more.

Common Input Elements

HTML provides several input elements that can be included within a form:

  • <input type="text">: Creates a single-line text input field.
  • <input type="password">: Creates a password input field with hidden characters.
  • <input type="checkbox">: Creates checkboxes that allow users to select multiple options.
  • <input type="radio">: Creates radio buttons, allowing users to select one option from multiple choices.
  • <select> and <option>: Create dropdown lists with selectable options.
  • <textarea>: Creates a multi-line text input field for longer text.

Example Form

Here's an example of an HTML form:

                    
<form action="/submit" method="post">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username">
    
    <label for="password">Password:</label>
    <input type="password" id="password" name="password">
    
    <label>Select your gender:</label>
    <input type="radio" id="male" name="gender" value="male">
    <label for="male">Male</label>
    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Female</label>
    
    <label for="country">Country:</label>
    <select id="country" name="country">
        <option value="usa">USA</option>
        <option value="canada">Canada</option>
        <option value="uk">UK</option>
    </select>
    
    <label for="comments">Comments:</label>
    <textarea id="comments" name="comments"></textarea>
    
    <input type="submit" value="Submit">
</form>
