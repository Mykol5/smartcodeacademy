Git and GitHub Assignment

Assignment: Git Basics and GitHub Integration

Your task is to follow the steps outlined in the git lecture and perform the following actions:

Task 1: Setting Up Git

Task 2: Basic Git Commands

Task 3: GitHub Repository Creation

Create a new repository on GitHub following the steps outlined in the lecture. Take a screenshot of your GitHub repository page. Save the image as "github_repository.jpg".

Task 4: Connecting Git with GitHub

Connect your local Git repository with the GitHub repository you created. Take a screenshot of your terminal showing the execution of the Git commands for this process. Save the image as "git_github_connection_terminal.jpg".

Task for Submission

Complete all tasks as stated above, take a screenshot of your terminal showing the Git commands you executed, and another screenshot of your GitHub repository. Save the images as "git_intro_terminal.jpg" and "git_intro_github.jpg" respectively, and submit them for assessment.

Result
