Loops in Python

Lecture: While Loop

The `while` loop continues to execute a block of code as long as a specified condition is `True`. It's particularly useful when you don't know in advance how many iterations are required.

Syntax of a While Loop:


while condition:
    # Code to execute as long as the condition is True

Lecture: For Loop

The `for` loop is used to iterate over a sequence (such as a list, tuple, or string) or other iterable objects. It executes a block of code for each item in the sequence.

Syntax of a For Loop:


for variable in sequence:
    # Code to execute for each item in the sequence

Class Example: Using Loops

Let's use both `while` and `for` loops to count from 1 to 5 and print each number.


# Example of using a while loop to count from 1 to 5
count = 1
while count <= 5:
    print(count)
    count += 1  # Increment the count

# Example of using a for loop to count from 1 to 5
for count in range(1, 6):
    print(count)
