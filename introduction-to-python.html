Lecture: What is Python?

Welcome to the introduction to Python! Python is a versatile and easy-to-learn programming language. It is widely used in various domains, including web development, data analysis, machine learning, and more. Python is known for its readability and simplicity, making it an excellent choice for beginners.

Python uses variables to store data. A variable is like a container for data. In Python, you can create a variable and assign data to it. For example, you can create a variable called message and store a text message in it:


        message = "Hello, World!"
        print(message)  # Output: Hello, World!

In this example, message is the variable name, and "Hello, World!" is the data stored in the variable. We use the = sign to assign data to a variable. Python supports various data types, including:

  • Strings: Used for text, enclosed in single or double quotes. Example: "Hello, World!"
  • Integers: Used for whole numbers. Example: 42
  • Floats: Used for decimal numbers. Example: 3.14159
  • Booleans: Used for true or false values. Example: True or False
  • Lists: Used to store collections of items. Example: [1, 2, 3]
  • Dictionaries: Used to store key-value pairs. Example: {"name": "John", "age": 30}

Example:


        # Variables with different data types
        name = "John"  # String
        print(name)
        age = 30       # Integer
        print(age)
        height = 1.75  # Float
        print(height)
        is_student = True  # Boolean
        print(is_student)
        grades = [85, 90, 78]  # List
        print(grades)
        person = {"name": "Alice", "age": 25}  # Dictionary
        print(person)

In the example above, we've defined variables with various data types in Python and used the print function to display their values. We'll explore these data types and more Python concepts in this course.
